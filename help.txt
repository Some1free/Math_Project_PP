EN
This is a program designed for simple calculations of bearing reactions in a statically determinate beam.
Configure the input file, or use the "config.cfg" file.
The default output file name is "results.txt".
You can change the input file, the output file and request to show the calculation.
You will do this using the arguments below:

-h or --help to show help
-i or --input to change the input file name
-o or --output to change te output file name
-c 		to save calculations
